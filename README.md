# Geometry Dash Music Downloader

Быстрый скачиватель музыки для Geometry Dash с кешированием на сервере и прогресс-баром.

## Особенности

- 🚀 **Быстрое скачивание** - музыка кешируется на сервере
- 📊 **Прогресс-бар** - видите процент загрузки в реальном времени
- 🎨 **Цветной интерфейс** - красивый вывод с colorama
- 💾 **Автоматическое кеширование** - повторные скачивания мгновенны
- 🧹 **Чистый интерфейс** - экран очищается после каждого скачивания

## Установка

### Требования
- Python 3.7+
- pip

### Зависимости

```bash
pip install flask flask-cors requests pysocks colorama
```

## Использование

### Запуск сервера

На машине с доступом к Newgrounds (или через прокси):

```bash
python server.py
```

Сервер запустится на `http://0.0.0.0:5000` и будет кешировать музыку в папке `sound/`.

### Запуск клиента

На клиентской машине:

```bash
python client.py
```

Введите ID музыки из Newgrounds и программа скачает её с прогресс-баром.

Музыка сохраняется в `%localappdata%\GeometryDash\`

## Как это работает

1. **Первый запрос** - клиент запрашивает музыку у сервера
2. **Сервер проверяет кеш** - если музыка уже есть в папке `sound/`, отправляет её
3. **Если нет в кеше** - сервер скачивает с Newgrounds и кеширует
4. **Клиент получает файл** - с красивым прогресс-баром и сохраняет локально

## Структура

```
.
├── server.py      # Сервер с кешированием
├── client.py      # Клиент для скачивания
├── sound/         # Папка кеша на сервере
└── README.md      # Этот файл
```

## API Сервера

- `GET /download/<music_id>` - скачать музыку по ID
- `GET /health` - проверка здоровья сервера

## Конфигурация

### Сервер (server.py)

Измените `SOUND_PATH` для другой папки кеша:

```python
SOUND_PATH = os.path.join(os.path.dirname(__file__), "sound")
```

### Клиент (client.py)

Измените `SERVER_URL` для подключения к другому серверу:

```python
SERVER_URL = "http://your-server:5000"
```

Измените `SAVE_PATH` для другой папки сохранения:

```python
SAVE_PATH = os.path.join(LOCALAPPDATA, "GeometryDash")
```
